Setup and Data Preparation

First, we will recreate the Riding Mower dataset. This is a classic “Two Populations” problem with two predictors: Income (\(x_1\)) and Lot Size (\(x_2\)).

# Load necessary libraries

library(MASS)    # For lda() and qda()

#2. Linear Discriminant Analysis (LDA)

LDA assumes \(\Sigma_1 = \Sigma_2 = \Sigma\) (equal covariance matrices) and multivariate normality.

# Fit the LDA model
lda_model <- lda(Owner ~ Income + LotSize, data = mowers)


# Display the coefficients (Fisher's linear combination)

3. Evaluating Performance: APER and Confusion Matrix

We use the Confusion Matrix to calculate the Apparent Error Rate (APER).

# Predictions on the training set
lda_pred <- predict(lda_model, mowers)
conf_matrix <- table(Actual = mowers$Owner, Predicted = lda_pred$class)

print(conf_matrix)
##           Predicted
## Actual     Nonowner Owner
##   Nonowner       10     2
##   Owner           1    11
# Calculate APER
 
aper <- (conf_matrix[1,2] + conf_matrix[2,1]) / sum(conf_matrix)
cat("Apparent Error Rate (APER):", round(aper * 100, 2), "%\n")
## Apparent Error Rate (APER): 12.5 %
  1. Lachenbruch’s Holdout Procedure (Leave-One-Out CV)

The lda() function has a built-in parameter CV = TRUE to perform the “holdout” procedure described in your notes.

# Perform Leave-One-Out Cross-Validation
lda_cv <- lda(Owner ~ Income + LotSize, data = mowers, CV = TRUE)

# Confusion Matrix for Holdout Procedure
cv_matrix <- table(Actual = mowers$Owner, Predicted = lda_cv$class)
print(cv_matrix)
##           Predicted
## Actual     Nonowner Owner
##   Nonowner        9     3
##   Owner           2    10
# Estimated Actual Error Rate (AER)
aer_hat <- (cv_matrix[1,2] + cv_matrix[2,1]) / sum(cv_matrix)
cat("Estimated AER (Holdout):", round(aer_hat * 100, 2), "%\n")
## Estimated AER (Holdout): 20.83 %
  1. Quadratic Discriminant Analysis (QDA)

When the assumption \(\Sigma_1 = \Sigma_2\) is violated, we use QDA.

# Fit QDA model
qda_model <- qda(Owner ~ Income + LotSize, data = mowers)

# Predict for a new observation (e.g., Income=100, LotSize=19)
 
new_obs <- data.frame(Income = 100, LotSize = 19)
predict(qda_model, new_obs)

  1. Fisher’s Method (Multi-class Example) For more than two populations, we can visualize the separation using the first two discriminants.
# Using the built-in Iris dataset for multiple populations (3 species)

data(iris)
iris_lda <- lda(Species ~ ., data = iris)

# Projecting data onto the first two discriminants (LD1 and LD2)
lda_values <- predict(iris_lda)
plot_data <- data.frame(lda_values$x, Species = iris$Species)
